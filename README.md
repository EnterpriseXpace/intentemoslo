# Intentémoslo de Nuevo

Plataforma de diagnóstico emocional de pareja.

## Requisitos previos

- Node.js (Version LTS recomendada)
- npm

## Instalación

1. Instalar dependencias:
   ```bash
   npm install
   ```

2. Ejecutar entorno de desarrollo:
   ```bash
   npm run dev
   ```

3. Abrir [http://localhost:3000](http://localhost:3000)

## Estructura

- `/src/app`: Rutas (App Router)
- `/src/components`: Componentes UI
- `/src/styles`: Estilos globales y tokens
- `/docs`: Documentación del proyecto (No borrar)
- `/reference_views`: Vistas de referencia (No borrar)

## Reglas (Ver /docs)

- No rediseñar.
- Usar Tailwind con los colores definidos en configuración.
- Mantener la coherencia visual.
